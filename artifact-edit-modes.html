Artifact Edit Mode Options

Three approaches for indicating navigation depth when editing artifacts

Option 1: Slide-Over with Peek Edge
Workspace slides over chat, leaving a visible edge. Click/swipe the edge to return to chat. Pros: Visual metaphor of layering. Cons: Slightly less workspace.
